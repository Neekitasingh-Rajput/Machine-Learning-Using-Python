# Machine-Learning-Using-Python 🤖📊

-----

## Project Description

This repository contains the **lab exercises, datasets, and lab test implementations** completed for the **Machine Learning** course during **Semester IV** at **CHRIST (Deemed to be University)**.

The work focuses on building a foundational understanding of machine learning workflows, including **Exploratory Data Analysis (EDA)**, **data preprocessing**, **model building**, and **model evaluation** using Python.

-----

## Datasets

The datasets used in this repository are stored in the `datasets/` directory.

### Included Datasets
- **students.csv** – Used for EDA and feature engineering  
- **StoreSales.csv** – Used for aggregation and visualization  
- **student_exam_pass_logreg.csv** – Used for Logistic Regression  
- **hospital_admissions.csv** – Used for predictive modeling  
- **SMS Spam Dataset** – Used for Naive Bayes classification  

### External / Built-in Datasets
- **salaries.csv** – External dataset used for Decision Tree (not included)  
- **Iris Dataset** – Loaded using `sklearn.datasets`

## How to Use This Repository

To run the notebooks locally:

### Prerequisites

- **Python 3.x**
- **Jupyter Notebook / JupyterLab**

Install required libraries:

```bash
pip install numpy pandas matplotlib seaborn scikit-learn
```

### Steps to Run

1. Clone the repository:
```bash
git clone https://github.com/Neekitasingh-Rajput/Machine-Learning-Using-Python.git
cd Machine-Learning-Using-Python
```

2. Launch Jupyter:
```bash
jupyter notebook
```
3. Open any .ipynb file and run the cells.
